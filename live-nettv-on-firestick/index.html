How to Install Live NetTV on Firestick using Downloader

  • Power up your Amazon Firestick via USB and Connect to an external display using HDMI.
  • Currently, you will be on the Home Screen. Navigate through the Menu until you reach the Settings.
  • Under Settings, Select the Device. Then Tap on the Developer Options.
  • Two options are listed under Developer Options – ADB Debugging and Apps from Unknown Sources.
  • To install 3rd party apps that are not available in Amazon App Store you need to enable the “Apps from Unknown Sources“.
  • Tap on it and a popup message will appear, Click on the Turn On to completely enable unknown sources installation.
  • Return to Home Screen and Navigate to the Search icon.
  • By using the On-screen keyboard to type in the “Downloader” or the Voice Search [ Press and 🎙 hold to Voice Search]
  • It will display the search results for downloader, Find the Downloader App which is an orange-colored tile.
  • To ensure before installing the app is developed by Aftvnews. Tap on the Download button and wait until it gets installed.
  • Once finished, Tap on the Open button. You will be greeted on the Downloader App. Tap on OK.
  • Go to Settings on the left menu to check if APK Auto-Install is enabled so that it automatically ask to install APK files if downloaded.
  • Now go back to Home and In “Enter the URL of the website you want to load or the file you want to download” field, Type the official site URL and Tap on Go button.
  • Now the Download progress will begin, It might take few mins based on your Internet Speed.
  • Since APK Auto-Install is enabled, it will directly take you to the Installation page.
  • Read the File Permission for Privacy and Device Access then scroll down and Select INSTALL button.
  • The Screen displays “Installing..” and once completed it turns into “App Installed.”
  • Now Select OPEN to launch the app on firestick.
